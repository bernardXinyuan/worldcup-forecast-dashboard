# 版本路线图与 Backlog

## 1. 版本路线图

### V0.1 规划版

目标：完成需求、协作机制和部署规划。

状态：进行中。

交付：

- 项目总规划。
- PRD。
- Agent 工作流。
- 上下文交接规范。
- 部署与上线交付规划。

### V0.2 原型版

目标：完成可评审的页面原型和数据结构。

交付：

- 首页看板原型。
- 赛程预测页原型。
- 小组模拟器原型。
- 淘汰赛路径图原型。
- 球队详情页原型。
- 数据字典。

### V0.3 MVP 开发版

目标：本地可运行，支持示例数据。

交付：

- 前端应用。
- 示例赛事数据。
- 预测数据结构。
- 内容配置文件。
- 分享页面。

### V0.4 内测版

目标：部署测试环境，支持小范围内测。

交付：

- 测试环境。
- 内测链接。
- QA 测试报告。
- 内容配置说明。
- 反馈收集机制。

### V1.0 正式版

目标：正式上线并可公开访问。

交付：

- 生产环境。
- 监控告警。
- SEO 和分享配置。
- 发布和回滚流程。
- 内容运营日历。

## 4. 当前优先决策

1. MVP 是否包含用户投票。
2. 技术栈选择。
3. 数据源选择和授权策略。
4. 预测模型先采用规则模型、Elo 模型，还是接入外部预测数据。
5. 是否从 MVP 开始做 SEO 和分享卡片。
