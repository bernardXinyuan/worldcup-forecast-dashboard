# 产品 PRD

## 1. 背景

2026 年世界杯由美国、加拿大、墨西哥联合举办，赛事周期长、关注度高、球队数量扩展后赛程和出线形势更复杂。用户需要一个既能快速浏览预测结论，又能互动模拟比赛结果的独立预测看板。

## 2. 目标与成功指标

### 产品目标

- 让用户在 10 秒内看懂今日赛事、热门球队和冠军概率。
- 让用户可以通过模拟器探索小组出线和淘汰赛路径。
- 让预测结果具备可解释性，而不是只给一个概率数字。
- 让产品可以在世界杯赛前上线，并在赛中持续运营。

### 成功指标

- 首页首屏加载时间小于 3 秒。
- 移动端核心页面可用，无关键内容遮挡。
- 用户可以在 30 秒内完成一次比分模拟。
- 每个预测页面都展示更新时间和数据说明。
- 数据更新失败时有可见兜底状态。

## 3. 用户场景

### 场景 A: 快速看预测

用户打开首页后看到今日比赛、冠军概率、热门球队和关键悬念。

### 场景 B: 查看单场比赛

用户进入比赛详情，查看胜平负概率、预测比分、关键因素和置信度。

### 场景 C: 模拟小组赛

用户修改尚未结束比赛的比分，系统实时重算积分榜、出线概率和排名变化。

### 场景 D: 研究淘汰赛路径

用户查看不同小组结果下的淘汰赛潜在对阵，以及每支球队晋级概率。

### 场景 E: 分享预测结果

用户把冠军概率榜、单场预测或模拟结果分享到社交平台。

## 4. 功能范围

### REQ-001 首页综合看板

优先级：P0

描述：展示冠军概率榜、今日比赛预测、热门球队、最大悬念和更新时间。

验收标准：

- 用户进入首页后首屏可看到冠军榜和今日比赛。
- 每个预测模块显示概率、趋势和更新时间。
- 无数据时显示友好兜底状态。

### REQ-002 赛程与单场预测

优先级：P0

描述：展示所有比赛，支持按日期、阶段、球队筛选，并进入单场预测详情。

验收标准：

- 支持小组赛、淘汰赛、决赛筛选。
- 单场详情包含胜平负概率、预测比分、关键因素和置信度。
- 比赛结束后显示实际比分和预测命中状态。

### REQ-003 小组积分榜

优先级：P0

描述：展示各小组积分、净胜球、进球数和出线概率。

验收标准：

- 小组积分规则正确。
- 排名变化和出线概率可见。
- 支持比赛结束后更新真实积分。

### REQ-004 小组赛模拟器

优先级：P0

描述：用户可以修改尚未结束比赛的比分，系统实时重算积分和出线形势。

验收标准：

- 修改比分后积分榜即时更新。
- 显示每队小组第一、小组第二和出局概率。
- 支持一键恢复默认预测。

### REQ-005 淘汰赛路径图

优先级：P0

描述：展示从小组赛到冠军的预测路径和晋级概率。

验收标准：

- 路径图在桌面端可完整阅读。
- 移动端提供可横向滚动或分阶段浏览的布局。
- 每支球队显示进入下一轮概率。
- 支持基于小组结果更新对阵路径。

### REQ-006 球队详情页

优先级：P0

描述：展示球队基础信息、分组、赛程、实力指标、晋级概率和预测趋势。

验收标准：

- 每支球队有独立详情页。
- 可查看该队小组赛和潜在淘汰赛路径。
- 可查看冠军概率和趋势变化。

### REQ-007 分享能力

优先级：P0

描述：支持分享首页、单场预测、球队详情和模拟结果。

验收标准：

- 页面 URL 可直接分享并恢复对应状态。
- 分享卡片标题、描述和图片可配置。
- 模拟结果至少支持通过 URL 参数或本地状态生成分享视图。

### REQ-008 数据来源与可信说明

优先级：P0

描述：展示数据来源、更新时间、模型说明和免责声明。

验收标准：

- 所有预测页面显示更新时间。
- 提供数据源说明入口。
- 明确预测仅供娱乐和参考，不涉及博彩建议。

### REQ-009 用户投票

优先级：P1

描述：用户可以为比赛结果或冠军球队投票。

验收标准：

- 同一用户或设备有基础防刷策略。
- 页面可展示用户投票分布。
- 运营可关闭投票入口。

### REQ-010 内容管理

优先级：P1

描述：提供内容运营配置首页推荐、专题文案、分享卡片和热门球队的能力。

验收标准：

- 运营不依赖开发即可修改核心文案。
- 支持预览发布效果。
- 支持回滚到上一版本配置。

## 5. 信息架构

- 首页看板
- 赛程预测
- 小组赛
- 淘汰赛路径
- 球队详情
- 数据说明
- 分享结果页
- 内容管理

## 6. 数据需求

### 基础数据

- 球队信息。
- 分组信息。
- 赛程信息。
- 比赛结果。
- 场馆和开球时间。

### 预测数据

- 胜平负概率。
- 预测比分。
- 出线概率。
- 晋级概率。
- 冠军概率。
- 置信度。
- 预测变化原因。

### 内容数据

- 首页推荐模块。
- 热门球队配置。
- 专题标题和文案。
- 分享卡片配置。

### 互动数据

- 投票对象。
- 投票时间。
- 匿名用户标识。
- 来源页面。

## 7. 非功能要求

- 首页首屏加载时间小于 3 秒。
- 移动端适配常见手机屏幕。
- 后端或静态数据异常时有兜底内容。
- 页面核心文案支持中文，保留未来英文扩展能力。
- 不展示博彩诱导文案。
- 记录关键错误日志和数据更新时间。
- 支持基础 SEO 和社交分享预览。

## 8. 合规与免责声明

- 预测内容仅供娱乐和赛事讨论。
- 不提供下注建议，不接入博彩交易。
- 数据源必须确认授权或使用可合法引用的数据。
- 用户投票数据尽量匿名化，避免收集不必要个人信息。
